Это настоящее руководство для пользования программой "ProgramTkani"

1. При запуске программы необходимо пройти авторизацию: выбрать роль, ввести логин и пароль, верно переписать капчу*.
2. В зависимости от вашей роли, вам будет доступно либо просто просмотр каталога товара, либо полный доступ к таблице из базы данных.
